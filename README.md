**✨ GlowGuide AI**
**Your Personal Beauty & Skincare Assistant**
GlowGuide AI is an AI-powered customer support chatbot built using Streamlit for the beauty and skincare industry.
It simulates real-world e-commerce support flows including:
📦 Order Tracking
🔄 Returns & Refund Processing
💳 Payment Assistance
📋 FAQ Automation
👩‍💼 Human Escalation Simulation

**Features**
Multi-step conversational flows
Order ID detection logic
Return reason validation
Escalation to human agent
Custom pink & gold themed UI
Session-based memory using Streamlit state

**Tech Stack**
Python
Streamlit
Session State Management
Rule-Based NLP Intent Detection
